3. Purpose of PRE tag

Pre Tag stands for preformatted text. It displays the text exactly as it was written in HTML code.

3.1 Pre tag differs from P tag

The pre tag in HTML preserves spaces and line breaks, displaying text exactly as written in the code.
 In contrast, the p tag is for regular paragraphs, where spaces and line breaks are rendered as a single space.
